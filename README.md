Commands to convert radixes.

* x2b - convert to binary
* x2o - convert to octal
* x2d - convert to decimal
* x2h - convert to heaxadecimal
* x2x - convert any radix to any radix

## Install ##

### PyPI ###

    pip install x2x

### From source ###

    python setup.py install
